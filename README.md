# Лабораторная работа №4: Работа с коллекциями и CSV

## Описание проекта

Данный проект представляет собой приложение для парсинга CSV-файла, содержащего информацию о сотрудниках. Программа считывает данные из файла `foreign_names.csv`, преобразует каждую строку в объект `Person` и собирает их в коллекцию `List<Person>`.

Проект демонстрирует навыки работы с файлами, внешними библиотеками (opencsv), а также правильное использование Java коллекций и принципов объектно-ориентированного программирования.

## Ключевые возможности

*   **Объектно-ориентированное моделирование:** Данные представлены в виде четких Java-классов (`Person`, `Division`, `Gender`).
*   **Парсинг CSV:** Для надежного разбора файла используется библиотека `opencsv`.
*   **Корректное преобразование типов:** Строковые данные из файла преобразуются в соответствующие типы Java: `int`, `LocalDate`, `enum`.
*   **Эффективная работа с объектами:** Реализован механизм кэширования для объектов `Division`. Это гарантирует, что для каждого уникального подразделения создается только один объект в памяти, на который ссылаются все соответствующие сотрудники.
*   **Надежная работа с ресурсами:** Используется `try-with-resources` для автоматического и безопасного закрытия файловых потоков.
*   **Автоматизированное тестирование:** Качество и корректность работы парсера подтверждена с помощью Unit-тестов (JUnit 5).

## Технологии

*   Java 24 (Amazon Corretto)
*   Maven - для управления зависимостями и сборкой
*   OpenCSV - для парсинга CSV-файлов
*   JUnit 5 - для модульного тестирования

## Как собрать и запустить

Для работы с проектом вам понадобится установленный JDK и Maven.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Syxarik2005/java-lab4.git
    cd java-lab4
    ```

2.  **Сборка проекта:**
    ```bash
    mvn compile
    ```

3.  **Запуск тестов:**
    Эта команда запустит все Unit-тесты, которые доказывают корректность работы парсера.
    ```bash
    mvn test
    ```

4.  **Запуск приложения:**
    Эта команда запустит основной класс `Main`, который прочитает большой файл и выведет первые 10 записей.
    ```bash
    mvn exec:java -Dexec.mainClass="com.sergey.Main"
    ```
